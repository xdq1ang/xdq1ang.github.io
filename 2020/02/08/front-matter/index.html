hexo博客的md文件的front-matter用法

Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量，举例来说：

下是预先定义的参数，您可在模板中使用这些参数值并加以利用。

参数 描述 默认值
layout 布局
title 标题 文章的文件名
date 文件日期 文件的创建日期
updated 更新日期 文件更新日期
comments 文章评论功能 true
tags 标签（不用于分页）
categories 分类（不用于分页）
permalink 覆盖文章网址
images 文章简要缩略图
description 文章简要（显示read more按钮）
